# 🍜 上海餐厅打卡清单

## 🌶️ 川菜 / 辣系

> 个人偏好品牌：平川、南兴园、柴门、邓记、坛、四吉、地瓜（digua）

### 💰 平价
- [ ] **九老汉** — 烤鱼 + 蛋花汤（蛋花汤是白的，味道很正）
- [ ] **惠中川香蛙**
- [ ] **川香煲**
- [ ] **三喜担担面**
- [ ] **重庆麻辣烫**（五角场国权路5号口那家）

### 💰💰 中等价位
- [ ] **竹屋** — 全上海少见的青椒炒猪耳朵
- [ ] **贾温柔**
- [ ] **不吃素**

### 💰💰💰 精致 · 有环境 + 创意
- [ ] **地瓜（digua）** — 四川老板开的自贡菜
- [ ] **鸟鸟川菜馆** — 很辣，以爆炒为主

### 🔥 火锅
- [ ] **付二娘美蛙鱼**
- [ ] **小鲭楼美蛙鱼**
